Bovenstaande deployment diagram geeft een beeld hoe de applicatie moet gedeployed worden. Spotitube webapplicatie artifact dea_spotitube_war wordt gedeployed op JEE7 Complient Container. De applicatie server communiceert met de webbrowser via HTTP waarmee hij aantal requests kan uitvoeren. Verder is er ook een communicatie tussen database server en de applicatie, in dit geval MariaDB, en deze communicatie vindt plaats door middel van TCP/IP. Met de source code is er een SQL-script meegeleverd onder resources map waardoor de tabellen aangemaakt kunnen worden en benodigde test data voor deze tabellen ingevuld kunnen worden. De playlists, users, tracks tabellen bevatten benodigde properties van de betreffende dto.
